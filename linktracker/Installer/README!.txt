Please!

Before using the Installer please just in case make sure
you don't have any of these following tables in your MySQL
Database that you will be using:

-Users
-Links
-Tracking

+ secondly when using the Installer make sure you haven't
edited /class/config.php file, if you have then please
make a backup of it.

- If something does go wrong or the Installer does not work
you will have to fill out /class/config.php file manually.
make sure all the locations are correct.
